# Tesserae: A Collection of Low-Level Micro-Coding Experiments

> "Tesserae" (noun) — Individual tiles used to create a mosaic.

Welcome to my personal low-level tech sanctuary. This repository serves as a growing mosaic of my journey towards mastering computer graphics, theoretical physics, and extreme executable size optimization (Size-coding).

Every project here is a standalone, hand-optimized experiment built from scratch, aiming to push the absolute boundaries of hardware and mathematical optimization.

---

## 📂 Project Index

### 🎨 [01-Primus-1K: The Kaleidoscope](./01-primus-1k)
* **Concept:** Pure C + Win32/OpenGL fixed pipeline. Blits 28 independent, dual-axis rotating, color-shifting quads directly onto the screen via raw Windows Desktop DC blasting.
* **Size Constraint:** **EXACTLY 1,024 bytes** (Compressed via Crinkler).
* **Core Micro-Optimizations:** Full `CRT-free` compilation, strategic IAT (Import Address Table) reduction, and bit-level floating-point loop tuning.
